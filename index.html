S3 Orchestrator (Go / PostgreSQL / AWS SDK / Docker)

S3-compatible proxy that combines multiple storage backends into a unified endpoint. Routes objects across providers (OCI, Backblaze B2, AWS, MinIO, etc) with per-backend quota enforcement, automatic overflow routing, and cross-backend replication. Features SigV4 authentication with multi-bucket credential isolation, multipart upload support, monthly usage limits, a three-state circuit breaker for database outages, and a built-in web dashboard with an interactive object tree browser. Metadata lives in PostgreSQL; all queries generated by sqlc. Includes OpenTelemetry tracing and Prometheus metrics.
